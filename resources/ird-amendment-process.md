## Purpose
This procedure explains how to lodge an IRD amendment for a client.

## When to Use
Use this process when a filed return needs correction or adjustment.

## Steps

1. Log into the IRD portal.
2. Locate the client profile.
3. Select the correct tax year.
4. Submit the amendment request.
5. Confirm submission reference number.

## Important Notes

- Always double-check the IRD number.
- Confirm the correct tax year before submitting.
- Record amendment reference in CRM.
